Welcome to Firefly (Fly-Ko-Lib) documentation!
==============================================

**Firefly** and its Kotlin implementation **Fly-Ko-Lib** is a top-level 2D game framework focusing on an intuitive API and build upon stringent architecture and design principles like Component-Entity-System, Builder/DSL and component indexing for fast access.

Follow this tutorial to learn how to setup a project with Firefly and learn all about its components.

Or take a look at the <a href="https://github.com/AndreasHefti/flyko-lib">Firefly Github Repository</a>

.. note::

   This project is under active development.
